## Giới thiệu

GitHub Pages cho phép triển khai website tĩnh miễn phí từ GitHub.

## Cách hoạt động

- Đẩy code lên GitHub
- Kích hoạt Pages
- Website chạy online

## Kết luận

GitHub Pages rất phù hợp để host blog học tập.
